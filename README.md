# LeaveFlow – Employee Leave Management System

LeaveFlow is a modern Employee Leave Management System designed to simplify leave application, approval, and tracking within an organization.

It allows employees to apply for leave, view leave history, update pending requests, cancel requests before approval, and track leave status in real time.

HR/Admin can review all leave requests, approve or reject applications, monitor pending approvals, and manage the complete leave workflow through a centralized dashboard.

---

## CRUD Operations for Leave Requests

### Create

Employees can create a new leave request by selecting leave type, leave dates, and providing a reason for leave.

### Read

Employees can view their leave history and track leave request status. HR/Admin can view all employee leave requests from the dashboard.

### Update

Employees can update leave requests only before HR approval. Once approved or rejected, the request cannot be modified.

### Delete

Employees can cancel or delete leave requests only while the request is still in Pending status.

## Features

### Employee Portal

* Secure Login / Registration
* Apply for Leave
* Select Leave Type
* View Leave History
* Update Pending Requests
* Cancel Leave Requests
* Track Leave Status (Pending / Approved / Rejected)

### HR/Admin Portal

* Secure Login
* View All Leave Requests
* Approve / Reject Requests
* Search and Filter Requests
* Dashboard Analytics
* Track Employee Leave Records

---

## Workflow

1. Employee logs in
2. Employee submits leave request
3. Request status becomes Pending
4. HR reviews the request
5. HR approves or rejects
6. Employee sees updated status instantly
